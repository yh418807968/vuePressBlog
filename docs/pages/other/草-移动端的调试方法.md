### vconsole
[官方地址](https://github.com/Tencent/vConsole)
```js
// 安装包
npm install vconsole
```
```
// 引入vconsole
import VConsole from 'vconsole'
// 实例化
window.vConsole = new VConsole()
```
最终会在右下角出现vconsole
